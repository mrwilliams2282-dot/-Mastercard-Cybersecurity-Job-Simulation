# -Mastercard-Cybersecurity-Job-Simulation

## 📋 Project Overview
This repository contains my deliverables and key takeaways from the Mastercard Cybersecurity Job Simulation completed via Forage. The program focused on practical, real-world tasks performed by Mastercard's security team, specifically targeting phishing analysis and organizational security awareness.

## 🛠️ Skills Demonstrated
*   **Phishing Email Analysis:** Inspected email headers, domains, and malicious links.
*   **Defense Strategy:** Designed simulated phishing campaigns to train employees.
*   **Data Interpretation:** Analyzed campaign metrics to identify organizational risk.
*   **Security Literacy:** Translated technical security risks into actionable business advice.

## 🧠 Simulation Modules & Tasks

### 1. Phishing Analysis
*   **Task:** Evaluated suspicious emails targeting company infrastructure.
*   **Action:** Analyzed email design, fake domains, and sender signatures to identify indicators of compromise (IoCs).
*   **Result:** Drafted clear technical notes outlining why the email was a threat and how to block it.

### 2. Security Awareness & Culture
*   **Task:** Designed a simulated phishing training campaign for internal teams.
*   **Action:** Created realistic phishing templates based on common social engineering tactics and established performance metrics (click rates, report rates).
*   **Result:** Developed a deployment strategy to improve the company's overall security posture.

## 🚀 Key Takeaways
*   Learned how a global financial enterprise approaches employee security training.
*   Gained experience balancing technical defense protocols with corporate policy.
